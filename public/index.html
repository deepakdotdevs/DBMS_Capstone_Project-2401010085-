SQL Queries Used

These are the exact queries from the project PDF running against your database:

-- Low Stock Check
SELECT p.name, i.quantity FROM inventory i
JOIN products p ON i.product_id = p.product_id
WHERE i.quantity < 50;
-- Total Orders by Distributors
SELECT d.name, SUM(o.quantity) AS total_orders
FROM orders o JOIN distributors d ON o.distributor_id = d.distributor_id
GROUP BY d.name;
-- Transportation Cost Analysis
SELECT transport_mode, SUM(cost) AS total_cost
FROM logistics
GROUP BY transport_mode;
